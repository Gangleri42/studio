    How this works
    

      This is an in-browser Coldcard simulator. The screen rectangle hosts
      a live <canvas> driven by the MicroPython firmware
      compiled to WebAssembly; the keypad buttons inject keycodes into
      the firmware loop. Tap ✓ to confirm, × to
      cancel. Keyboard 0–9, Enter,
      Esc, ←/→ work on both variants.
    

    

      "Share Seed Words via NFC" writes an NDEF tag image to a MEMFS
      file at /work/nfc-dump.ndef; the page reads it and
      hands the seed words to the SeedHammer engrave pipeline. Per the
      project's input-is-NFC-only rule, this is the same payload SeedHammer
      consumes from real hardware.
