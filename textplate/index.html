SeedHammer Text Plate

Compose free-form text for an 85 × 85 mm plate. The preview is the engraving.
1:1 0×0

How it engraves. A line break starts a new line; long lines wrap. The firmware engraves at the largest text size whose grid holds the composition: . The preview above always shows the size the firmware will pick.

Android Chrome: the Web NFC button compiles the composition to a seedhammer.com:curves vector record and writes it directly, no app needed. Phone (NFC Tools): copy, then in NFC Tools add a Text record, paste and write while the SeedHammer start screen shows; the firmware lays the text out itself. USB NFC writer: python3 cmd/textplate/write-nfc.py [--curves] plate.txt from the repository.
